### Useful VS Code abbreviations

* Type `html` in an html file and see a list of options to auto populate. See the outcome of this auto pop by holding `ctrl` + `space`.
* Auto populate a list by using `ul>li.item$*3` which would create an unordered list with three entries.
* To insert a certain number of words type `lorem` which will automatically insert 30 words. Type  `lorem22` to insert 22 words.
